What is HTML?

Hypertext markup Language, a standardized system for tagging text files to achieve font,color, graphics, and hyperlinl effects on World Wide Web pages. HTML describe the structure of a web page. HTML consists of a series of elements. HTML elements tell the browser hoe to diaplay the content.
HTML is heavily used for creating pages that are displayed on the world wide web. HTML elements are the building blocks of HTML pages.


What is Semantic html?

Semantic HTML, also known as semantic markup, refers to the use of HTML tags that convey the meaning or semantics of the content contained within them. A semantic element clearly describes its meaning to both the browser and developer By adding semantic HTML tags to your pages, you provide additional information that helps define the roles and relative importance of the different parts of your page.
